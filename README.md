# MARL Framework for Scalable Multi-Agent Decision Systems

A Multi-Agent Reinforcement Learning (MARL) framework built with PyTorch, Gymnasium, and MAPPO (Multi-Agent Proximal Policy Optimization), implementing Centralized Training and Decentralized Execution (CTDE) for cooperative decision-making, simulation-based learning, and large-scale real-world optimization. It supports training, experimentation, and AI model development.

This framework explores how multiple agents interact within simulated environments to learn coordinated policies through MAPPO-based centralized critic optimization, decentralized execution, PPO-style policy updates, trajectory management, experiment tracking, visualization, and automated evaluation.
